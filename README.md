# EcoWatt India: Premium Energy Analytics Platform

EcoWatt India is a high-performance, AI-driven energy auditing dashboard built with **Streamlit**. It allows users to track household appliance consumption, compare monthly usage trends, and receive intelligent saving tips to reduce both electricity bills and CO2 footprints.

## Live Demo
Experience the live application here: [ecowattindia.streamlit.app](https://ecowattindia.streamlit.app/)

## Key Features
- **Smart Audit Engine:** Real-time calculation of units (kWh), monthly cost (INR), and CO2 emissions based on appliance wattage.
- **Deep Comparison Mode:** Side-by-side analysis of usage trends between "Last Month" and "Current Month" to identify consumption spikes.
- **AI-Powered Insights:** Automatically identifies "culprit" appliances and suggests specific optimization strategies.
- **Premium UI:** Custom glassmorphism design using CSS transitions, Google Fonts (Orbitron & Exo 2), and interactive Plotly visualizations.

## Tech Stack
- **Framework:** [Streamlit](https://streamlit.io/)
- **Data Visualization:** [Plotly](https://plotly.com/python/)
- **Data Manipulation:** [Pandas](https://pandas.pydata.org/)
- **Styling:** Custom CSS3 with Glassmorphism and Responsive Layouts
